## ğŸš€ Quick Start

### Local Development

1. Clone the repository:
```bash
git clone https://github.com/aviral-bhardwaj/avi.git
cd avi
```

2. Open `index.html` in your browser:
```bash
# On macOS
open index.html

# On Windows
start index.html
```

3. Or use a local server (recommended):
```bash
# Using Python 3

# Using Node.js (http-server)
npx http-server

# Then open http://localhost:8000 in your browser
```
